Sonar LED Cube

Github

4x4x4 RGB LED cube with built-in sonar sensors to allow for interactive user motion controls.

The LEDs are positively driven by layer and individually addressed to ground by using TCL5940 LED drivers and P-CH MOSFETs. Components are controlled by an Atmega328P-PU microcontroller, the same one used in popular Arduino Uno boards, and the embedded logic is programmed in C++ using the Arduino IDE.

There are two modes of operation: animation mode and game mode. The animation mode will cycle through a sequence of programmed animations. The game mode is a reflex-based game where the user will use the sonar sensors as a input to stop an led racing through the cube. The objective is to stop the led by quickly moving your hand away from sonar sensor so the led lands at a specific spot in the cube.

Skills used/gained:
  • Arduino programming in C++
  • Analog and digital circuit design
  • Soldering

LED Word Clock

Github

Sleek digital LED clock displaying the time in words.

The LEDs individually addressed by using ULN2003AN drivers controlled by 74HC595N shift registers. Components are controlled by a custom breadboard Arduino with an Atmega328P-PU microcontroller. The clock is programmed in C++ using the Arduino IDE and everything is housed together in an IKEA picture frame.

  • Arduino programming in C++
  • Analog and digital circuit design
  • Soldering
